The Mobile Key visualisation (vis6)

This visualisation is a multi-access key with a 'mobile-first' design. Taxa, represented by rectangles, are ranked according to how well they match any character states you specify, with the best overall matches appearing at the top.

Each taxon is represented by a rectangle with two numbers in the top-right. The first number indicates how many of the currently specified characters 'match' the values in the knowledge-base for the taxon. (In this case 'match' means any positive score.) The second number shows the overall matching score (overall 'for' score minus the overall 'against' score). These are also coloured to give a quick visual reference: blue indicates the best overall matching score and vermilion the worst, whilst an overall matching score of zero is coloured pale yellow. Intermediate scores are shown with intermediate colours. Taxa are ranked in each list according to their scores with the best matches appearing at the top.

If you click the question mark icon to the right of the scores, a page is shown that gives more detail on how each score was calculated.

Three filters above the rectangles representing the taxa allow you to filter the taxa. The first of these - All - displays all taxa. The second - Matched - limits the dsplay to only those taxa where the number of matching characters is the same as the number of characters specified by the user. The final one - Discounted - limits the display to only those taxa where the number of matching characters is less than the number specified by the user.

As you specify character states, those taxa appearing at the top of the 'evidence balance positive' column (on the left) are those which are the most likely matches. The more states you specify, and the higher this score, the more reliable the match is.

Character input

Specifying values for character input is intutive. If the character input panel is not visible, either tap the keyboard icon, or swipe right from the left-hand side of the display. The input panel can be 'pinned' by tapping the pin icon, or removed by either swiping left or tapping the back arrow.

Species accounts

Tapping on a taxon name invokes a tabbed species accounts page that can include the following tabs:

  • Knowledge-base values - details of the knowledge, e.g. character state values, for that taxon in the knowledge-base underlying the key.
  • Images - any images provided for the taxa are available here.
  • NBN Atlas - distribution maps downloaded from the NBN Atlas via a web service. (This tab is not shown if the knowledge-base developer has not provided links to the NBN Atlas.)
  • Info - supplementary textual information about the taxa, e.g. species descriptions.

Using as an offline app

This key is a 'Progressive Web App' (PWA) which brings two advantages:

  1. It can be installed and used like a app on a mobile phone, for example being invoked from an app icon on the homescreen.
  2. It can be used offline - i.e. where there is no internet connection.

Installing as an app

This app can be run from any mobile device that supports a modern internet browser, such as Chrome, by navigating to the web address in the browser in the usual way. On platforms with full support for PWA's you will be offered the option of adding an icon for the app to the homescreen.

Once installed on Android, the icon will be visible in your apps, just as if you had installed from the Play Store. As when installing from the app store, you may then have to move the icon to your homescreen.

Support for PWAs on iOS normally lags a little behind Android and the experience on iPhone will depend on your iOS version, but should be similar.

Using offline

To use the app offline, you need to first download the apps resources before going into the field whilst you still have an internet connection. Invoke the main menu by tapping the menu icon and then select the 'Offline options' item towards the bottom of the menu.

Tap the 'Download for offline use' button to download all images and text files associated with the app. Note that NBN distribution maps, if they are included with the app, cannot be bulk-downloaded in this way, but any that you load individually whilst in range of wifi will remain available to you when offline.
