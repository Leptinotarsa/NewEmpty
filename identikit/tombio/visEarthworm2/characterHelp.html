Epilobic head
Tanylobic head

The first segment on an earthworm, also called the head, consists of the peristomium and the prostomium. The prostomium is a sort of flap over the mouth projecting forward from main part of the first segment (the peristomium). From the dorsal side of the earthworm the prostomium also projects backwards through the peristomium towards the boundary between first and second segments.

Where this projection ends before the boundary with the second segment, the head is said to be epilobic. Where it reaches the boundary with the second segment, it is said to be tanylobic.

Close up of the male pore

The male pore an is a more or less obvious opening visible as a slit, sometimes with swollen lips. The segment on which this falls can be used as a character in the key. For some species (e.g. Lumbricus sp.) it can be very hard to see.

Sometimes the swellings around the male pore encroach onto the previous segment and/or the segment after. For the purposes of the key, it is the segment that the actual pore falls on that you must note.

In some species, e.g. Lumbricus sp., the male pore can be very difficult to see.

Clitellum

The position of the clitellum, also called the saddle, is a key identification feature. The position is specified by the number of the segment, counted from the head end, on which the clitellum starts and the number of the segment on which the clitellum ends.

Care is necessary when deciding which segment the clitellum starts on. Sometimes the clitellum covers more segments on the dorsal side of the earthworm than it does on the ventral side, in which case it is the coverage on the dorsal side that matters. Because a well developed clitellum often masks the segments that it covers, it is sometimes hard to discern the segment that it finishes on. In such cases, it can help to view the ventral side of the earthworm since the segmentation is often less masked here.

In this multi-access key, you can specify the length of the clitellum - i.e. the number of segments it covers - even if you can't discern the start and end segments, which can be the case when the earthworm is damaged before the clitellum. If you know the start and end segments, it is best to specify these rather than the length, since they provide information on both position and length. If you've specified the start and end segments, there's no need to specify the length.

Nonsensical values, e.g. clitellum end less than clitellum start, are shown in vermillion (red).

The clitellum develops on mature earthworms, so immature specimens may not have a discernible clitellum which makes identification difficult.

TP (tubercula pubertatis)

The TP, full name tubercula pubertatis, is situated on the clitellum. It normally appears as one, or a series of, raised areas. The actual structure is very variable between species. Sometimes it can be very difficult to see.

The segment on which the TP starts and that on which the TP ends are important identification features. The actual shape/structure of the TP is a useful confirmation feature for identification, but with one or two exceptions, is not diagnostic. In this multi-access key, the segments on which the TP starts and ends can be used as an ID character, but the TP shape can't (although it can be used as a visualisation character).

In some species of earthworms, the TP is visible before the clitellum develops and can sometimes help to identify even immature specimens.

Closely-paired setae
Widely-paired setae

Earthworms in the family Lumbricidae have four pairs of setae, also called chaetae, on each segment. Whether or not the setae in each pair are widely spaced or closely spaced is an important identification feature.

When looking for the character, it is important to look at the tail end of the earthworm - the setae on other parts can have different spacing to that indicated in the key.

Setae can be hard to see. Try adjusting the angle of the lighting and viewing against different colour backgrounds - e.g. both black and white.

The size of earthworms can be variable even within a single species, but there are some gross differences which can be useful identification characters.

In this multi-access key you can specify both the length and the diameter (i.e. the thickness) of the earthworm. Because of the variability of these characters it is best not to use them as diagnostic features on their own. But they are useful confirmation characters and may be particularly useful where other morphological characters are not discernible.

All measurements are in millimetres (mm).
