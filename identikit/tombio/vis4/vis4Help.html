The Full taxon details visualisation (vis4)

This visualisation is a fast way to explore the taxonomic knowledge-base for any given species. It displays all information including character state values, images and text files side-by-side for a species selected from a drop-down list.

To use the visualisation, just select a taxon from taxon selection control (see below) and all information in the knowledge base for this taxon will be displayed on the right, including the character state values, images (if any referenced) and any other information in referenced HTML files. An example is shown below.

You can tailor the display to exclude any of these three types of information (character state values, images or text) by unchecking the relevant check-boxes (also visible in the image below).

Context menu

The context menu for this tool has a single option - Get URL for full details view. Selecting this menu item builds a URL that can be used to reproduce the page as you have configured it. For example displaying the selected taxon and image. The context menu, can be invoked by right-clicking the mouse with the cursor anywhere in this visualisation.
